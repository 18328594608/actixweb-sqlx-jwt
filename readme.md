## md5 api server

1. mod.js server
`./mod.js -d`

2. mock.js test: 
`./mock.js -- -d`

### [Compiling Executables](https://deno.land/manual@v1.7.4/tools/compiler)

```sh
deno compile --unstable --allow-net --allow-read=./ --allow-run -o mod mod.js
```
